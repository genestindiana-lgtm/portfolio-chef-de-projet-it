# Guide de rédaction SEO :

Ce document présente les règles à respecter pour la rédaction d’articles et de métadonnées optimisées pour le référencement naturel (SEO).


## Méta-titre :

- Un méta-titre doit contenir **60 caractères maximum**
- Au-delà de cette limite, le titre est **tronqué par Google**, ce qui réduit sa lisibilité et peut impacter le référencement

### Règles de rédaction

- Placer **le mot-clé principal en premier**
- Ne **pas mettre de point final** à la fin du méta-titre (cela compte comme un caractère inutile)

## Méta-description

- Longueur recommandée :
  - **Minimum : 120 caractères**
  - **Maximum : 160 caractères**

Une méta-description trop courte manque d’information, tandis qu’une description trop longue sera coupée dans les résultats de recherche.

### Règles de rédaction :

- Résumer clairement le contenu de l’article
- Inclure le mot-clé principal
- Ne **pas mettre de point final** à la fin de la phrase



## Déterminer le mot-clé central :

Chaque article doit être construit autour d’un **mot-clé principal**.

### Exemple :
Pour un article sur le règlement de **P’tites Figouzes** :
- Mot-clé central : `P’tites Figouzes`
- Ce mot-clé doit apparaître **en premier dans le méta-titre**


## Cas particulier : articles sur le Festival JdR :
Pour tous les articles ayant pour sujet le festival :

**Règle :**
Festival JdR doit être placé en premier dans le méta-titre



## Articles sur un exposant ou un artiste 

### Nomenclature à respecter pour le méta-titre :
Nom de la personne – domaine d’activité


### Exemple :
Tournesol Dupont – Auteur de bande dessinée


### Contenu attendu dans la méta-description :

La méta-description doit inclure :
- la présence de la personne au Festival JdR
- sa spécialité ou sa notoriété
- une œuvre connue si applicable

### Exemple :
Tournesol Dupont au Festival JdR, rencontrez l’auteur de OEUVRE à LIEU DATE


## Longueur minimum des articles :

Un article web doit contenir au minimum : 300 mots


Cela permet :
- une meilleure indexation par les moteurs de recherche
- un contenu jugé suffisamment riche par Google



## Règles typographiques importantes 

### À éviter dans les métadonnées :

- Ne pas ajouter de point final (`.`) dans :
  - les méta-titres
  - les méta-descriptions

Chaque caractère compte dans les limites imposées par les moteurs de recherche.

##Check list SEO

### Contenu :
- [X] Le titre contient le mot-clé principal en début de phrase
- [X] Le premier paragraphe mentionne le sujet clairement
- [X] L'article fait au moins 300 mots
- [X] La date et le lieu sont rappelés en accroche 

### Rappel :
- **Méta-titre** : renseigné, 50-60 caractères
- **Méta-description** : renseignée, 150-160 caractères
- **Slug** : propre et lisible (ex: `/jeux-de-role/fantasy`)
- **Image principale** : uploadée avec une description alt
- **Catégorie** : sélectionnée

### Avant de cliquer sur "Publier" :
- [x] Aperçu vérifié sur mobile
- [x] Orthographe vérifiée
- [x] Liens internes ajoutés (vers d'autres pages du site si pertinent)
