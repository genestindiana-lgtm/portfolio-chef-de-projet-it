# Contexte du projet — Festival JdR

Les noms ont été changé par souci de discrétion. Je précise que mon rôle a été celui de chargé de projet web / consultant digital. 

Dans le cadre d'un festival de fantasy avec une section dédiée aux jeux de société, un formulaire d'inscription à des tables de jeux de rôle a été mis en place.

Auparavant, cela se passait par une prise de contact avec le responsable du pôle qui rentrait sur un tableur Excel le nom des Maîtres du Jeu, ainsi que les futurs joueurs en les positionnant sur des créneaux de 3h de 10h à 18h pendant le week-end du festival.

Pour améliorer la fluidité des échanges, un formulaire d'inscription a été mis en place via le framework **Symfony**, découpé ainsi :

- Prise de contact par les **Maîtres du Jeu** pour créer la fiche du Jeu de Rôle
- Les Jeux de Rôle étaient classés par genre (fantasy, science-fiction, familiale, etc.) avec une image de couverture et le résumé de l'univers
- Les joueurs pouvaient parcourir ce catalogue pour se décider sur quel jeu de rôle se positionner
- Pour l'inscription des joueurs, un simple **pseudo** était demandé — dans l'objectif de stocker le moins d'informations personnelles possible

Ce formulaire a ensuite évolué vers un site web à part entière créé via Symfony : **Festival JdR**.



## Objectifs :

- Créer un **CMS avec Symfony** pour Festival JdR
- Créer un **panneau administrateur adapté à des utilisateurs peu habitués au numérique**, comprenant :
  - La création de catégories & d'articles
  - L'hébergement d'images
  - Des spécificités SEO (méta-titre, méta-description, etc.)
  - Une vue d'ensemble sur les tables de jeux de rôle encore disponibles
- Porter le formulaire initialement hébergé sur le précédent site vers le nouveau site Festival JdR
- Valoriser Festival JdR pour son **grand retour en tant que festival fin 2026**
- Améliorer le **référencement SEO** du site
- Enseigner à un public peu habitué au numérique à **rédiger pour le web**
- Avoir un site **responsive et accessible**
- Positionner le site sur la **Google Search Console**



## Contenu attendu :

- Un site accessible, qui charge rapidement et responsive
- Un **top 3 en positionnement SEO**
- Un formulaire d'inscription aux tables de Jeux de Rôle
- Une plateforme dédiée au concours de peinture sur figurines


## Cibles :

| Profil | Description |
|--------|-------------|
| **Rédacteurs** | Devant naviguer sur le panneau administrateur de manière fluide |
| **Public familial** | Voulant découvrir le jeu de société le temps d'un week-end |
| **Éditeurs, auteurs, boutiques** | Voulant promouvoir leurs œuvres le temps du salon |
| **Maîtres du Jeu & joueurs** | Créateurs de tables et participants |
| **Peintres sur figurines** | Voulant participer au concours **P'tites Figouzes** |



## Contraintes :

- Équipe réduite
- Mots-clés difficiles à mettre en avant — trop génériques
- Site web devant sortir vite : **±6 mois calendaire**



## Exclusions :
