# User Stories – Festival JdR

## Contexte

Le site du Festival JdR a pour objectif de faire découvrir le jeu de rôle au grand public, de valoriser les exposants et de permettre aux visiteurs de s’inscrire aux tables de jeu pour 
découvrir le jeu de rôle. Les Maîtres du Jeu peuvent créer leurs tables pour des sessions de 3h répartis de 10h à 18h le temps du festival. 

Les user stories ci-dessous ont été définies afin de formaliser les besoins fonctionnels du site du point de vue des utilisateurs finaux.


## US01 – Comprendre le festival

**En tant que** visiteur  
**Je veux** comprendre rapidement ce qu’est le Festival JdR  
**Afin de** savoir si l’événement peut m’intéresser

### Critères d’acceptation
- Une page d’accueil présente  festival
- Les informations essentielles (date, lieu, principe) sont visibles sans scroller excessivement
- Le contenu est accessible à un public familial

---

## US02 – Découvrir les exposants

**En tant que** visiteur  
**Je veux** consulter la liste des boutiques, éditeurs et artistes présents  
**Afin de** préparer ma visite et identifier les stands qui m’intéressent

### Critères d’acceptation
- Un menu avec chaque type d'expoisant : éditeurs, boutiques, illustrateurs, peintres sur figurines
- Chaque exposant dispose d’une description courte avec leurs réseaux sociaux 
- Les informations sont structurées et lisibles sur mobile

---

## US03 – Ouvrir une table de jeu de rôle durant le festival

**En tant que** visiteur et Maître du Jeu  
**Je veux** créer une table de jeu via un formulaire en ligne  
**Afin de** réserver ma place à une session de jeu de rôle

### Critères d’acceptation
- Le formulaire est accessible depuis le site
- Les champs obligatoires sont clairement identifiés
- Un message de confirmation s’affiche après l’envoi du formulaire

## US04 – Jouer à du Jeu de Rôle

**En tant que** visiteur et rôliste  
**Je veux** découvrir le jeu de rôle 
**Afin de** réserver ma place à une session de jeu de rôle

### Critères d’acceptation
- Le formulaire est accessible depuis le site
- Le formulaire permet de découvrir plusieurs JdR classés par type (fantasy, science-fiction, type de public etc)
- Les champs obligatoires sont clairement identifiés
- Un message de confirmation s’affiche après l’envoi du formulaire


## US05 – Découvrir le jeu de rôle avant de venir

**En tant que** débutant  
**Je veux** lire des articles expliquant ce qu’est le jeu de rôle  
**Afin de** me sentir plus à l’aise avant de participer au festival

### Critères d’acceptation
- Les articles sont accessibles depuis le menu principal
- Les contenus sont rédigés dans un langage accessible aux débutants
- Les pages sont optimisées pour le référencement naturel (SEO)

## US06 – Participer au concours de figurines 

**En tant que** amateur de peinture sur figurine

**Je veux** participer au concours de peinture sur figurine

**Afin de** remporter un prix, et de recevoir des conseils auprès du jury

### Critères d’acceptation
- Le concours est visible et accessible depuis la page d'accueil 
- Le réglement est lisible et segmenté selon les types de figurines 
- Les noms du jury sont identifiables 
- Les pages sont optimisées pour le référencement naturel (SEO)
